Co-op Cinema

Watch movies together with friends, anywhere in the world

💫 What is this?

A synchronized video player that lets you watch movies with remote friends in real-time. Both viewers control the same playback - play, pause, and seek together!

🎯 How to Use

1
Create or Join Room

Start a new theater or join with a room code

2
Share Room Code

Copy the code and send it to your friend

3
Load a Video

Paste a YouTube, Vimeo, Twitch, or Dailymotion link, a direct video URL, or open a local video file

4
Watch Together!

Play, pause, and seek - everything syncs automatically

📝 Note: Online videos sync automatically for all viewers. For local files, both viewers need the same video file — nothing is uploaded.
